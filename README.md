# 教育信息管理系统 python353

## 项目概述

教育信息管理系统是基于Python Django框架开发的应用，旨在为教育机构提供全面的信息管理解决方案。系统通过不同的角色权限，为教师、学生和管理员提供各自的功能模块，实现教学资源的有效管理和高效沟通。

## 技术栈

- **后端框架：** Django
- **数据库：** MySQL
- **开发工具：** PyCharm等Python开发工具
- **前端UI框架：** Bootstrap

## 功能模块

### 系统角色

- 学生
- 教师
- 管理员

### 学生功能

1. 课表查询
2. 课程信息操作
3. 作业提交与管理
4. 班级信息管理
5. 视频学习
6. 笔记编写
7. 内容收藏
8. 个人中心操作

### 教师功能

1. 课表查询
2. 班级管理
3. 作业发布与管理
4. 签到发布
5. 通知发布
6. 视频发布
7. 个人中心操作

### 管理员功能

1. 教师信息增删改查
2. 书籍信息增删改查
3. 实验室管理
4. 工资管理
5. 教室管理

## 运行环境

- Python环境
- MySQL数据库服务
